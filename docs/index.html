25th Oct 2017 - 2nd Nov 2017

Welcome

Data Analysis Club (DAC) is a student-run organisation in Singapore Institute of Management that thrives on the desire to instill the passion for data, create and sustain a community that fosters collaborative learning and act as a platform for students to meet like-minded peers. We ultimately aim to empower the student body with data analysis skills and create a data-positive environment for data-enthusiasts to thrive.

This is an E-Book written by the Curriculum Director (2017) of SIM DAC for the Data Analysis with R Workshop as well as for our members who requires additional help in the field of Data Analytics.

Data Analysis with R workshop (DAWR) is catered to the students of SIM to better equip them with the necessary skills to start off with basic Exploratory Data Analysis (EDA).

Topics covered:

  1. R101
  2. Data Visualisation
  3. Data Transformation
  4. Exploratory Data Analysis

The free open source tool, R, is utilised.

To download both R and rstudio, you can visit the following websites.

  • https://www.r-project.org/
  • https://www.rstudio.com/products/rstudio/download/

For more information about our club, do vist our website at https://simdaclub.com/ as well as our github page at https://simdac.github.io/.
